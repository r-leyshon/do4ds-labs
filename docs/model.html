1 Get Data

Code 
con = duckdb.connect("my-db.duckdb")
df = con.execute("SELECT * FROM penguins").fetchdf().dropna()
con.close()

2 Define Model and Fit

Code 
X = get_dummies(df[['bill_length_mm', 'species', 'sex']], drop_first = False)
y = df['body_mass_g']

model = LinearRegression().fit(X, y)
